# Hyprland keybindings

`$mod` = Super (Win key)

## Applications

| Binding | Action |
|---|---|
| `$mod Return` | Terminal (Ghostty) |
| `$mod Space` | Launcher (Fuzzel) |
| `$mod B` | Browser (Chrome) |
| `$mod Shift E` | Editor (Zed) |

## Windows

| Binding | Action |
|---|---|
| `$mod Q` | Close window |
| `$mod V` | Toggle floating |
| `$mod F` | Fake fullscreen |
| `$mod Shift F` | Real fullscreen |
| `$mod P` | Toggle pseudo-tile |
| `$mod X` | Toggle split direction |

## Focus

| Binding | Action |
|---|---|
| `$mod ←↑↓→` | Move focus |

## Move window

| Binding | Action |
|---|---|
| `$mod Shift ←↑↓→` | Move window in layout |
| `$mod Alt ←↑↓→` | Resize window |
| `$mod LMB drag` | Move floating window |
| `$mod RMB drag` | Resize floating window |

## Workspaces

Home row maps to workspaces 1–9.

| Binding | Action |
|---|---|
| `$mod A–L` | Switch to workspace 1–9 |
| `$mod Shift A–L` | Move window to workspace 1–9 |
| `$mod scroll` | Cycle workspaces |

```
A  S  D  F  G  H  J  K  L
1  2  3  4  5  6  7  8  9
```

## Scratchpad

A hidden overlay workspace — send windows there and toggle visibility.

| Binding | Action |
|---|---|
| `$mod Z` | Toggle scratchpad |
| `$mod Shift Z` | Send window to scratchpad |

## Clipboard

| Binding | Action |
|---|---|
| `$mod C` | Open clipboard history picker |

## Screenshots

| Binding | Action |
|---|---|
| `Print` | Copy region to clipboard |
| `Shift Print` | Save region to file |
| `$mod Print` | Copy full screen to clipboard |

## Media & brightness

| Binding | Action |
|---|---|
| `XF86AudioRaiseVolume` | Volume +5% |
| `XF86AudioLowerVolume` | Volume −5% |
| `XF86AudioMute` | Toggle mute |
| `XF86AudioMicMute` | Toggle mic mute |
| `XF86MonBrightnessUp` | Brightness +5% |
| `XF86MonBrightnessDown` | Brightness −5% |
| `XF86AudioPlay` | Play/pause |
| `XF86AudioPrev` | Previous track |
| `XF86AudioNext` | Next track |

## Notifications & session

| Binding | Action |
|---|---|
| `$mod N` | Dismiss notification |
| `$mod Shift N` | Dismiss all notifications |
| `$mod Escape` | Lock screen |
| `$mod Shift M` | Exit Hyprland |
